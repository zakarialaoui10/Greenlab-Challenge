## Features 
 - [x] Change font family & size and color
 - [x] Local storage 
 - [x] Upload Background 
 - [x] Export Data from Excel File 
 - [x] Orbit and transform control 
 - [x] Preview of generated Attestations
 - [x] Download Attestations as pdf file
